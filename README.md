# Lyvia Health Ltd — CQC Governance Hub

This repository contains the source files for the **Lyvia Health Ltd Clinical Governance Hub**, a CQC-compliant governance portal for a digital private dermatology service.

The hub is published via GitHub Pages and provides CQC inspectors, staff, and governance leads with structured access to all governance documentation, policies, meeting records, and key contacts.

---

## About Lyvia Health Ltd

Lyvia Health Ltd is a digital private dermatology service registered with Companies House. The service operates as a remote, adults-only (18+) clinic issuing private prescriptions only. CQC registration is currently in progress.

| Detail | Information |
|---|---|
| Service Type | Digital Private Dermatology |
| Delivery | Remote / Online Only |
| Clinical Platform | Semble |
| Patient Scope | Adults 18+ only |
| Prescribing | Private prescriptions only — no NHS FP10/EPS, no controlled drugs, no oral isotretinoin |
| CQC Status | Registration In Progress |
